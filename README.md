# cc-lab

Claude Code plugins by [Lionel Di Giacomo](https://github.com/melek).

## Plugins

| Plugin | Description |
|--------|-------------|
| [lcars](https://github.com/melek/lcars) | Self-correcting cognitive ergonomics for Claude Code |
| [workplanner](https://github.com/melek/workplanner) | Daily work-planning and progress-tracking for Claude Code |
| [stepwise](https://github.com/melek/stepwise) | Autonomous systematic literature review for Claude Code |
| [proven](https://github.com/melek/proven) | Verification-informed thinking for everyday development |

## Installation

Add the marketplace:

```bash
claude plugins marketplace add melek/cc-lab
```

Install a plugin:

```bash
claude plugins install lcars@cc-lab
claude plugins install workplanner@cc-lab
claude plugins install stepwise@cc-lab
claude plugins install proven@cc-lab
```
