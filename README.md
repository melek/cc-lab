# melek-claude-code-plugins

Claude Code plugins by [Lionel Di Giacomo](https://github.com/melek).

## Plugins

| Plugin | Description |
|--------|-------------|
| [lcars](https://github.com/melek/lcars) | Self-correcting cognitive ergonomics for Claude Code |
| [workplanner](https://github.com/melek/workplanner) | Daily work-planning and progress-tracking for Claude Code |
| [stepwise](https://github.com/melek/stepwise) | Autonomous systematic literature review for Claude Code |

## Installation

Add the marketplace:

```bash
claude plugins marketplace add melek/melek-claude-code-plugins
```

Install a plugin:

```bash
claude plugins install lcars@melek-claude-code-plugins
claude plugins install workplanner@melek-claude-code-plugins
claude plugins install stepwise@melek-claude-code-plugins
```
